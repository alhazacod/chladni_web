¿Qué son las figuras de Chladni?

Las figuras de Chladni son patrones que se forman cuando partículas como arena o sal se colocan sobre una superficie que vibra. Al aplicar sonido o vibraciones, se generan zonas donde la superficie no se mueve (líneas nodales), y es allí donde las partículas se acumulan, formando patrones geométricos sorprendentes. Este experimento permite observar la relación entre el sonido, la frecuencia y las formas físicas.
